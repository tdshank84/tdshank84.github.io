# Harper's STEM Learning v0.9.1

GitHub Pages-ready princess castle web app.

## Landscape mode

The app is designed for landscape play. The web app manifest requests landscape mode when installed to the Home Screen. If Safari or iOS does not allow a hard rotation lock, the app shows a full-screen "Turn iPad Sideways" message in portrait.

## Install on iPad

1. Upload these files to GitHub Pages.
2. Open the GitHub Pages URL in Safari.
3. Tap Share.
4. Tap Add to Home Screen.
5. Launch from the Home Screen icon.

## Saved stars

Star count is saved in the browser with localStorage under the same key, so updating files should not erase progress.
