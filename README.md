# Harper's STEM Learning

GitHub Pages-ready web app.

## Upload
1. Upload everything in this folder to a public GitHub repo.
2. Go to Settings → Pages.
3. Source: Deploy from branch.
4. Branch: main, folder: /root.
5. Open the Pages URL in Safari and choose Share → Add to Home Screen.

## Star Count
Stars are saved in browser localStorage using `harpersStemPrincessStars.v2`, so progress remains after refreshes and app updates as long as the same browser/site URL is used.
