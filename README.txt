Harper's STEM Learning - Web App Package

This package is ready for GitHub Pages, Cloudflare Pages, Netlify, or any static web host.

Files:
- index.html: the game
- manifest.webmanifest: installable web app metadata
- service-worker.js: offline cache for hosted HTTPS sites
- icons/: Home Screen icons

Install on iPad/iPhone:
1. Host these files online. GitHub Pages is fine.
2. Open the hosted URL in Safari.
3. Tap Share.
4. Tap Add to Home Screen.
5. Tap Add.

Important:
- Add to Home Screen works from Safari using a hosted URL.
- It will not install correctly from iCloud Files preview.
- Voice requires tapping Start Voice once because iOS requires user interaction before speech audio.
